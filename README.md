## Project setup

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

# Crear un recurso (--no-spec para no incluir test)

```
nest g res pokemon --no-spec
```

# Docker

```
docker-compose up -d
```
